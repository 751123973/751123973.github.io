sigmoid 函数

$\sigma(z) = {1 \over {1 + e^{-z}}}$

值得注意的是 $\sigma’(z) = \sigma(z)(1 - \sigma(z))$

sigmoid 函数经常被用于二分类，当 $\sigma(z) < 0.5$ 时，被认为 $P = 0$。而 $\sigma(z) \ge 0.5$ 时，被认为 $P = 1$。

在 logistic regression 里，我们把每个特征乘于一个回归系数，然后相加，将得到的总和带进 $\sigma(z)$ 中，得到一个 $0 \sim 1$ 的值。

即 $z = \omega_{0}x_{0} + \omega_{1}x_{1} + \cdots + \omega_{n}x_{n}$，如果采用向量的写法可以写成 $z = w^{T}x$。

其中，$\omega$ 是回归系数（权重）的了列向量，而 $x$ 是输入的特征值列向量。

因此我们现在剩下的问题就是如何求解回归系数（权重 $\omega$）。

梯度上升(gradient ascent)


梯度

梯度是一个向量，表示某一函数在该点处的方向导数沿着该方向取得最大值，即函数在该方向变化最快(梯度的方向)，变化率最大(梯度的模)。

我们将梯度记为 $\nabla$，则 $f(x,y)$ 的梯度为 $\nabla f(x,y) = \begin{bmatrix}\partial f(x,y)\over \partial x \\ \partial f(x,y) \over \partial y \end{bmatrix}$ 。

表示我们现在需要沿着 $x$ 轴移动 $\partial f(x,y)\over \partial x$，沿着 $y$ 轴移动 $\partial f(x,y)\over \partial y$ 。($f(x,y)$ 要在这些点上有意义且可微)

但是我们上面的式子中，只给出了梯度的方向和变化率，并没有给出步长，因此我们还要定义一个步长 $\alpha$，$\alpha$ 也会被称之为学习率(learning rate)。
